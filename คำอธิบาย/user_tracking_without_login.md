# วิธีการเก็บชื่อคนเพิ่ม Usecase โดยไม่ต้องมีระบบ Login

หากคุณไม่อยากทำระบบ Login ที่ยุ่งยาก (ต้องมีอีเมล, รหัสผ่าน, ลืมรหัสผ่าน ฯลฯ) แต่ยังอยากรู้ว่า **"ใครเป็นคนเพิ่ม Usecase นี้"** วิธีที่เหมาะสมและเป็นที่นิยมที่สุดคือการใช้ **Local Storage ร่วมกับการถามชื่อเล่น (Nickname)** ครับ

---

## 💡 ไอเดียที่ดีที่สุด: "Guest + Local Storage"

หลักการทำงานคือ:
1. **เมื่อกดปุ่ม "Add Usecase" ครั้งแรก:** ระบบจะเช็กว่าในเครื่อง(เบราว์เซอร์)นี้ เคยบันทึกชื่อไว้หรือยัง
2. **ถ้ายังไม่มี:** เด้ง Popup เล็กๆ ขึ้นมาถามว่า *"กรุณาใส่ชื่อเล่นของคุณก่อนเพิ่ม Usecase"*
3. **บันทึกไว้ในเครื่อง:** พอยูสเซอร์พิมพ์ชื่อ (เช่น "มาร์ค") ระบบจะเอาชื่อนี้ไปจำไว้ใน `localStorage` ของเบราว์เซอร์
4. **ส่งไปพร้อมข้อมูล:** เมื่อกดยืนยันสร้าง Usecase ระบบจะเอาชื่อ "มาร์ค" ส่งแนบไปกับฐานข้อมูล (Supabase) ด้วย
5. **ครั้งต่อไปไม่ต้องกรอกแล้ว:** เพราะเบราว์เซอร์จำชื่อ "มาร์ค" ไว้แล้ว สามารถกดสร้าง Usecase ได้เลย

---

## 🛠️ สิ่งที่ต้องแก้ในโค้ดหากจะทำวิธีนี้

### 1. แก้ไขฐานข้อมูล (Supabase)
คุณต้องเข้าไปที่ Supabase แล้วเพิ่มคอลัมน์ใหม่ในตาราง `usecases`
- **ชื่อคอลัมน์:** `created_by` (หรือ `author`)
- **ชนิดข้อมูล:** `VARCHAR` หรือ `TEXT`
- **ตั้งค่าเริ่มต้น (Default):** อนุญาตให้เป็นค่าว่าง (Nullable) ได้ เผื่ออันเก่าที่ไม่มีชื่อ

### 2. แก้ไขฝั่งหน้าเว็บ (React)
**ส่วนของการเช็กชื่อ (Local Storage):**
```javascript
// ดึงชื่อจาก Local Storage
const savedName = localStorage.getItem('userNickname');

if (!savedName) {
   // ถ้าไม่มี ให้เด้ง Prompt หรือ Modal ถามชื่อ
   const name = prompt("กรุณาใส่ชื่อของคุณก่อนบันทึก:");
   if (name) {
      localStorage.setItem('userNickname', name);
   } else {
      return; // ถ้ายกเลิก ก็ไม่ให้บันทึก
   }
}
```

**ส่วนของการส่งข้อมูลไป Supabase:**
ในไฟล์ `UsecasesPage.jsx` ตอนสร้าง `newUsecase` ให้เพิ่มชื่อเข้าไปด้วย
```javascript
const newUsecase = {
   id: 'uc-' + Date.now(),
   title: usecaseData.title,
   description: usecaseData.description,
   image: usecaseData.image || null,
   date: new Date().toISOString(),
   created_by: localStorage.getItem('userNickname') // <--- เพิ่มบรรทัดนี้
};
```

**ส่วนของการแสดงผล (UI):**
ในการ์ด `Usecase` ก็แค่เพิ่มการแสดงผลชื่อลงไป
```jsx
<div className="usecase-meta">
   <span>👤 โดย: {uc.created_by || 'ไม่ระบุชื่อ'}</span>
   <span>📅 {new Date(uc.date).toLocaleDateString()}</span>
</div>
```

## 📌 สรุปข้อดี-ข้อเสีย ของวิธีนี้

**✅ ข้อดี:**
- ทำง่ายมาก ใช้เวลาแก้โค้ดไม่ถึง 10 นาที
- ยูสเซอร์ไม่ต้องรำคาญกับการสมัครสมาชิก
- รู้ได้ทันทีว่าใครเป็นคนสร้าง (ขึ้นอยู่กับความซื่อสัตย์ในการพิมพ์ชื่อ)

**❌ ข้อเสีย:**
- ถ้าเปลี่ยนคอมพิวเตอร์ หรือเปลี่ยนเบราว์เซอร์ มันจะถามชื่อใหม่
- ยูสเซอร์สามารถพิมพ์ชื่อปลอมหรือชื่อคนอื่นได้ (เพราะไม่มีการยืนยันตัวตน)

> [!TIP]
> **แนะนำ:** วิธีนี้เหมาะกับแอปพลิเคชันที่ใช้กันภายในทีม (Internal Tool) หรือแอปที่เปิดกว้างและเราแค่ต้องการเก็บเครดิตผู้สร้างคร่าวๆ ครับ หากคุณต้องการให้ผมเขียนโค้ดอัปเดตระบบนี้เข้าไปในโปรเจกต์เลย สามารถแจ้งได้เลยครับ!
